# MLB Bullpen Intelligence - Epic 1B

## Purpose
Epic 1B upgrades Bullpen Stress Index into a reliever-level production engine.

## Components
- Reliever Availability
- Reliever Fatigue
- Quality and Leverage
- Bullpen Hierarchy
- High-Leverage Availability
- Collapse Probability
- Composite BSI v2

## Output
The engine returns:
- availability_score
- fatigue_score
- quality_score
- leverage_score
- hierarchy_score
- collapse_risk_score
- final_bsi
- confidence
- warnings
- explainability

## Sample endpoints
- `/intelligence/mlb/bullpen/reliever/sample`
- `/intelligence/mlb/bullpen/hierarchy/sample`
- `/intelligence/mlb/bullpen/collapse/sample`
- `/intelligence/mlb/bullpen/bsi-v2/sample`

# NCAA Football Foundation - Epic 3A

## Purpose
Epic 3A starts NCAA Football production support by adding college-specific registry, schedule, roster, recruiting, transfer portal, and coaching-continuity intelligence.

## Components
- Team Registry
- Conference / Power Tier Classification
- Schedule Provider Interface
- Recruiting Profile
- Returning Production
- Transfer Portal
- Coaching Continuity
- Foundation Composite Engine

## Output
The foundation engine returns:
- roster_talent_score
- returning_production_score
- transfer_portal_score
- coaching_continuity_score
- program_strength_score
- volatility_score
- final_foundation_score
- confidence
- warnings
- explainability

## Sample endpoints
- `/intelligence/ncaaf/foundation/registry/sample`
- `/intelligence/ncaaf/foundation/schedule/sample`
- `/intelligence/ncaaf/foundation/foundation/sample`

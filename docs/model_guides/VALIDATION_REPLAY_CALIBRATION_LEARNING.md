# Validation, Replay, Calibration & Learning - Epic 1G

## Purpose
Epic 1G adds measurement infrastructure so LUCA can evaluate its own decisions.

## Components
- Replay Summary
- Calibration Buckets
- Brier Score
- Calibration Bias
- Feature Attribution
- Performance Slices
- Drift Detection
- Validation Report

## Output
The validation report returns:
- replay
- calibration
- attribution
- performance

## Sample endpoints
- `/validation/report/sample`
- `/validation/drift/sample`

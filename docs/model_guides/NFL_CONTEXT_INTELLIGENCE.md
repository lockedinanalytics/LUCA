# NFL Game Context Intelligence - Epic 2D

## Purpose
Epic 2D adds NFL situational, coaching, special teams, and environment intelligence.

## Components
- Situational Intelligence
- Coaching Intelligence
- Special Teams Intelligence
- Environment Intelligence
- Composite Context Engine

## Output
The engine returns:
- situational_score
- coaching_score
- special_teams_score
- environment_score
- game_script_projection
- pace_projection
- weather_risk
- hidden_yardage_edge
- late_game_management_edge
- final_context_score
- confidence
- warnings
- explainability
